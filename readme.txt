Gives you feed for Postqueues.

== Description ==

This Plugin provides a RSS-feed for every given [Postqueue](https://wordpress.org/plugins/postqueue/ "Postqueue Plugin").

== Installation ==

1. Upload `postqueue-feeds-wordpress.zip` to the `/wp-content/plugins/` directory
2. Extract the Plugin to a `postqueue-feeds` Folder
3. Activate the plugin through the 'Plugins' menu in WordPress
